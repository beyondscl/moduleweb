# moduleweb
#### 如何创建模块式项目
```
 1.新建maven空项目
 2.新建javase项目作为第一子项目，工具类引入
 3.新建web子项目，作为web的第一子项目，作为框架使用
 ```
 ```
 dogauth作为登录授权模块引入
 dogcore作为核心模块引入
 dogdb作为操作数据库，自动生成相关java jsp xml
 dogueditor百度富文本编辑器
 dogweb web主模块
 ```
#### 如何提交项目
```
 1.github新建仓库，复制地址
 2.idea允许CVS，add,commit，然后在push选择上一步的地址，push完成
```
